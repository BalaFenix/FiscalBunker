FiscalBunker
Proyecto Intermodular desarrollado durante el segundo curso del Grado Superior en Administración de Sistemas Informáticos en Red (ASIR).

Este proyecto surge de la colaboración con un compañero de formación. A pesar de compaginarlo con nuestra jornada laboral, dedicamos dos meses de investigación y trabajo técnico intensivo con el objetivo de consolidar y aplicar los conocimientos adquiridos durante estos dos años de estudio.

El resultado es un entorno de red virtualizado diseñado para la seguridad y monitorización. Somos conscientes de que existen áreas de mejora y nuevas funcionalidades por implementar, pero este repositorio recoge el trabajo realizado hasta la fecha.

Agradezco mucho la oportunidad de haber trabajado en este proyecto y el esfuerzo conjunto que nos permitió alcanzar estos resultados en un tiempo limitado.

Nota: Dentro del documento teórico, se incluye un enlace al anexo práctico con la guía detallada para la instalación y configuración de las herramientas utilizadas.
